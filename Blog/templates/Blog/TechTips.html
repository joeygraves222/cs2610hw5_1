My Git Cheatsheet

git init

Initializes a new repo

git status

Displays the status of the repo

git add 'filename'

Starts tracking the new file 'filename'

git add .

Starts tracking all of the new files

git commit

Commits the changes to the local repo

git commit -m"message here"

Commits the changes to the local repo with the message

git commit -am"message here"

Starts tracking all of the new files and commits with the message

git push

Pushes the commit(s) to the remote repo
